# 2.3 翻译功能

翻译功能，需要启用[互联网翻译](/docs/features/setting/translation#互联网翻译)或[AI 翻译](/docs/features/setting/translation#ai-翻译)。

若未启用以上配置，首次进入翻译功能界面时，将显示如下图的引导提示信息：

![默认](/img/23/default.png)

点击`立即配置`将会跳转到`设置`界面中的`翻译设置`。

## 互联网翻译

若启用了至少一个互联网翻译提供商，翻译界面将会展示为原文输入框以及翻译提供商信息。

![提供商](/img/23/providers.png)

左侧为原文输入框和语言选择列表；右侧为翻译提供商的翻译结果展示。

> 目前翻译语言只展示了常用的一些语言。

在左侧文本框中输入原文后，选择语言，点击翻译按钮，右侧将会展示翻译结果。如图：

![翻译](/img/23/translate.png)

启用的翻译提供商，翻译时会同步进行翻译，你可以选择最喜欢的翻译结果。点击复制按钮，即可将译文复制到剪贴板。

![复制结果](/img/23/copy.png)

## AI 翻译

要使用 AI 翻译功能，需要在设置中配置好[AI 翻译](http://localhost:3001/docs/features/setting/translation#ai-%E7%BF%BB%E8%AF%91)并启用。

启用 AI 翻译后，在翻译功能中就可看到 AI 翻译，如下图：

![](/img/23/ai.png)

> AI 翻译会消耗对应大模型的 Token，会产生相应的费用。

### 网页翻译

AI 翻译支持网页翻译。如果需要针对特定网页进行提示词或翻译选项调整，请先[设置 AI 翻译](http://localhost:3001/docs/features/setting/translation#ai-%E7%BF%BB%E8%AF%91)。

如果未设置 AI 翻译，当原文输入框输入 URL 地址后，界面会提示进行配置，如下图：

![](/img/23/no_ai.png)

设置 AI 翻译后，当原文输入框中输入 URL 地址后，翻译按钮将会展示文字 `AI`，右侧的互联网翻译提供商将会隐藏。如下图：

![](/img/23/url.png)

点击翻译按钮后，将开始翻译。

翻译逻辑如下：

1. 访问对应的 URL 获取网页全文
2. 使用 `AI 翻译` 中配置的`选择器`选取对应的网页内容
3. 使用`翻译源`选项来最终获取文本或 Html 作于翻译输入
4. 大模型使用`网页翻译提示词`进行翻译，最终输出译文

:::info 提示
右侧 `AI 翻译`中的 `Markdown` 开关默认开启。

开启时，如果输出的译文包含 Markdown 格式，界面上将会展示解析后的样式；关闭时，界面以纯文本格式展示。
:::
