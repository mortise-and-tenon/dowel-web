# 2.2.2 AI 设置

![](/img/22/222/ai.png)

AI 设置，目前支持 OpenAI 兼容的 API 大模型，如 `OpenAI`、`DeepSeek`、`Qwen` 等。用户需要先获取相关大模型的 API 以及密钥，才可以进行配置。

## 添加供应商

点击 `添加供应商` 按钮，界面展示如下图：

![添加供应商](/img/22/222/add.png)

填写你喜欢的名称后，点击`确定`保存。

界面上将展示添加的供应商列表，如下图：

![供应商列表](/img/22/222/list.png)

右侧表单中填写你的大模型 API 及密钥

- 点击 `检测` 按钮可以测试该 API 以密钥是否能成功访问大模型
- 点击 `保存` 按钮可以保存数据

保存后如下图所示：

![AI数据](/img/22/222/ai_data.png)

点击右上角的开关按钮，可以启用或停用该 AI 供应商，如下图所示：

![AI启用](/img/22/222/ai_open.png)
