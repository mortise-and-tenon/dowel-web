# 2.1 首页

## 界面布局

界面如下图：

![首页](/img/21/index.png)

软件整体布局：

- 顶部
  - 左上角：软件图标、当前功能页面标题
  - 右上角：白天/夜间模式切换、窗体最小化按钮、窗体最大化/还原按钮、窗体关闭按钮
- 左侧
  菜单栏，分别为：
  - 首页
  - 翻译
  - 设置
  - 帮助

## 首页内容

首页展示当前软件支持的翻译提供商列表信息。

左上角为提供商图标、名称以及启用状态，如厂商名称：`阿里云机器翻译`，若用户启用该提供商，将会展示绿色图标，如下图：

![](/img/21/index_open.png)

中间信息展示中，相关信息说明如下：

- `Token 用量`：用户当前（当月）已使用的翻译字符量，非准确统计，仅供参考。
- `Token 上限`：该提供商每月允许的翻译字符上限（软件给定默认值，用户可修改）。
